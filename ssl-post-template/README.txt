Some useful modifications for the AEI 10 m prototype logbook.

== Description ==

This adds editor templates to the `ssl-alp-inventory` post type to define some fields
that should always be present on such pages.

== Installation ==

1. Upload `ssl-post-template.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
